По TVI:

Чтобы алгоритм работал верно, необходимо назначить переменные получения температуры и установки,
Так как у меня это вынесено в отделньые датчики, то это выглядит так:

;#//Запрос требуемой температуры
proc fGetTt {} {
return [get_t "Каскад-гист"]
}


Для датчика цели, возможно в вашей системе потребуется заменить на 

return [- [get_hc_tns "Имя контура гидрострелки"] 2730]


;#//Запрос фактической температуры температуры
proc fGetCt {} {
return [get_t "Подача из гидрострелки"]
}

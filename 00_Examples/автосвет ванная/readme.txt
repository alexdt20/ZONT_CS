Сценарий позволяет настроить автоматизацию автоматического включения и выключения света в таких помещениях как ванная или туалет.

Для этого используются:
- датчик открытия двери
- датчик движения
- управляющий светом переключатель (например реле)


В сценарии не нужно ничего менять.
Нужно лишь пометить в названиях системы ZONT правильные назначения элементов для сценария

Для датчика движения в конце названия нужно дописать:
|ALM1
Для датчика открытия двери в конце названия нужно дописать:
|ALD1
Для элемента переключателя в конце названия нужно дописать:
|ALS1

цифра 1 - обозначет номер такой автоматизации.
Если их несколько то цифры соответственно должны отличаться.
Например, для датчиков и выключателя ванной они будут 1.
А для датчиков и выключателя туалета 2.


В сценарии есть настройки таймеров по умолчанию
AL_TDO 60
AL_TDC 11
AL_TM 60
AL_TL 600
AL_TCTR 180

Это значения времени в секундах:
AL_TDO - задержка выключения после открытия двери
AL_TDC - задержка выключения после закрытия двери
AL_TM - задержка выключения после обнаружения движения
AL_TL - задержка выключения после обнаружения движения при закрытой двери
AL_TCTR - задержка выключения, если было обнаружено включение света не по датчикам (т.е. если кто-то вручную включил свет и забыл выключить)


!!! Важный момент:
Параметр AL_TDC должен быть установлен на время большее чем время, за которое датчик движения покажет отсутствие движения.
Иначе после закрытия двери свет будет включен ложно по наличию движения.

В данном случае настроено 11 сек, т.к. zigbee датчик движения при обнаружении движения держит активное состояние 10 сек.


Для точной работы алгоритма, если используются zigbee датчики через шлюз zigbee2mqtt желатьельно в настройках шлюза установить для датчиков QOS = 1.
По умолчанию был 0.

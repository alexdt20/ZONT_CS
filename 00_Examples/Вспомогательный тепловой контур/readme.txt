

Сценарий позволяет организовать так называемый "вспомогательный" тепловой контур.
Например, для реализации управления отоплением в помещении по воздуху используется два контура:
Тёплый пол и Радиаторы
Тогда нагрев будет осуществляться тёплым полом, а когда он не справляется радиаторами.

Для этого сценарий устанавливает целевую контура радиаторы на 0,5 градуса нижэе чем у тёплого пола.

Для правильной работы сценария нужно задать названия контуров так:
Тёплый пол|=>Радиаторы
Радиаторы

Сценарий раз в секунду просматривает все контуры и найдя имя в котором есть символы |=> 
берёт после этих символов текст как имя вспомогательного контура
и устанавливает этому контуру уставку на пол градуса ниже.

(символ | позволяет добавить окончание названия, которое не будет видно пользователю)

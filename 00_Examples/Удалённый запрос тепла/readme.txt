
Два скрипта, позволяющие использовать удалённый запрос тепла с одного контроллера на другой.

Для работы необходимо в контроллере в котором находятсмя контуры потребителей создать вспомогательный
контур котла и назвать его boiler. В контурах потребителей настроить запрос тепла на этот бойлер.
В контроллере где находится котёл нужно создать вспомогательный контур протребителя с именем contur. 
И настроить его на работу по теплоносителю.

в контроллер с контурами потьребителями записать скрипт "удалённые потребители.pcl"
в контроллер с котлом прописать скрипт "удалённый котёл.pcl"

В скрипте контроллера с контурами потьребителями можно выбрать интерфейс по которому идёт обмен сообщениями.
Контроллер с котлом принимает сообщения с любого интерфейса.

-------

Для работы скриптов в контроллере должен быть записать вспомогательный файл скриптов utils.pcl
